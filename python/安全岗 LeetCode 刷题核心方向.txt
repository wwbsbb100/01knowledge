安全岗 LeetCode 刷题核心方向：







1. 基础算法（优先级最高）
安全场景中高频用到的基础逻辑，是面试必考点：

字符串处理：安全日志分析、数据包解析、漏洞特征匹配都依赖字符串操作

必刷题目：
简单：
LeetCode 344. 反转字符串、
LeetCode 28. 找出字符串中第一个匹配项的下标（KMP 算法入门）
LeetCode 125. 验证回文串
中等：
LeetCode 43. 字符串相乘（模拟网络数据进制转换）
LeetCode 151. 反转字符串中的单词（日志字段提取）

数组 / 切片操作：安全数据统计、流量特征提取常用
必刷题目：
简单：
LeetCode 26. 删除有序数组中的重复项、LeetCode 1 两数之和（基础逻辑）
中等：
LeetCode 56. 合并区间（安全事件时间范围合并）
LeetCode 15. 三数之和（数据关联分析）
哈希表（字典）：漏洞特征映射、日志计数、IP 黑名单匹配
必刷题目：
简单：
LeetCode 1. 两数之和
LeetCode 49. 字母异位词分组（特征归类）
中等：
LeetCode 146. LRU 缓存（安全缓存策略）、LeetCode 387. 字符串中的第一个唯一字符（日志唯一值提取）
2. 简单数据结构（优先级中）
安全场景中少量用到的基础结构，掌握核心逻辑即可：
栈 / 队列：漏洞扫描深度优先 / 广度优先遍历、数据包队列处理
必刷题目：
简单：
LeetCode 20. 有效的括号（语法 / 协议解析）LeetCode 232. 用栈实现队列
中等：
LeetCode 155. 最小栈（安全指标监控）
链表：仅需掌握基础操作，华为安全岗极少深考
必刷题目：
LeetCode 21. 合并两个有序链表
LeetCode 141. 环形链表（简单即可）
3. 安全场景相关算法（优先级中）
贴合安全工作的场景化题型，能体现你对安全 + 编程的结合理解：
模拟 / 暴力破解：模拟 DDoS 流量检测、简单漏洞扫描逻辑
必刷题目：
LeetCode 58. 最后一个单词的长度（日志字段提取）
LeetCode 66. 加一（IP 地址段计算）
简单递归 / 回溯：少量用于漏洞路径遍历（无需深钻）
必刷题目：
LeetCode 77. 组合（简单回溯）
LeetCode 104. 二叉树的最大深度（日志层级分析）
位运算：网络掩码计算、权限位处理（安全岗高频考点）
必刷题目：
LeetCode 191. 位 1 的个数（流量特征位统计）
LeetCode 231. 2 的幂（端口范围判断）、LeetCode 371. 两整数之和（无符号运算）
4. 无需重点刷的题型（优先级低）
华为安全岗几乎不考察，无需浪费时间：
动态规划（除了最简单的爬楼梯 LeetCode 70）
图论（除了简单的 BFS/DFS）
高级数据结构（红黑树、并查集等）
困难难度题目（仅基础题刷完后可少量尝试）
